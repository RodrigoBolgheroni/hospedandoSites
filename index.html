Passos para hospedar um Projeto no Github Pages

  1. Crie Um repostitorio Github com o mesmo nome do seu projeto.
  2. No terminal,inicialize o repostitorio local com git init.
  3. Altere o nome da branch principal main.
  4. Adicione os arquivos com git add . e faça o commit com git commit -m "first commit"
  5. Conecte ao repostitorio remoto com git remote add origin linkrepo.
  6. Envie os arquivos com git push -u origin main.
  7. No github,vá em Settings > Pages
  8. Em "branch" selecione main e a pasta / root clique em Save
  9. Após alguns segundos, seu site estara disponivel no link exibido em "Pages".
